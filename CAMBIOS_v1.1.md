# Chutemundo v1.1 - Estabilización

Cambios aplicados:

- Corregido el error del botón "Crear divisiones automáticas".
- Corregida la creación de torneos en formato "Copa con grupos".
- Corregida la creación de torneos en formato "Eliminación directa" para 2 a 8 equipos.
- Unificados los grupos de copa como "Grupo A" y "Grupo B".
- Agregada normalización para convertir torneos antiguos con "Grupo 1" y "Grupo 2" a "Grupo A" y "Grupo B".
- Agregado renderizado de "Cuartos de Final" en el detalle del torneo.
- Corregido el cálculo del ranking FIFA para aplicar recencia solo a los puntos del partido, no al total acumulado.
- Corregido el orden de carga de scripts para que Firebase esté disponible antes de iniciar la app.
- Cambiadas rutas de escudos desde absolutas a relativas para mejorar compatibilidad local y en subcarpetas.
- Corregidos textos con caracteres rotos por codificación.
- Agregados ajustes responsive para una navegación móvil más cómoda.

Notas:

- No se convirtió el proyecto a videojuego ni a ejecutable. Esta versión mantiene el enfoque como app web de gestión de torneos.
- No se modificó la base visual completa; se hizo una estabilización funcional y móvil.
