# Chute Mundo v1.3.1 — Publicador GitHub

Fecha: 5 de julio de 2026.

## Incorporado

- `Subir_Chute_Mundo_a_GitHub.bat`: ejecutable para Windows que automatiza `git add`, `git commit` y `git push`.
- `USO_PUBLICADOR_GITHUB.md`: instrucciones de instalación, uso y resolución de errores.

## Salvaguardas del publicador

- Verifica que Git esté instalado.
- Verifica que se use la carpeta original con `.git`.
- Muestra la rama y remoto configurados antes de publicar.
- Evita commits vacíos cuando no hay cambios.
- Configura el upstream de la rama si es necesario.

## Sin cambios funcionales

No se modificó la lógica del torneo, el historial, equipos, plantillas, copas, tablas ni estadísticas. Esta entrega solo agrega el flujo de publicación para GitHub.
